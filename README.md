# pasgen

A targeted password wordlist generator based on personal information patterns.

## Features

- Generate passwords using patterns: name+symbol+num, name+num+symbol, etc.
- Multiple case styles (UPPER, lower, Capitalized)
- Customizable number ranges
- Custom symbols
- Super fast generation using itertools

## Installation

```bash
git clone https://github.com/yourusername/pasgen.git
cd pasgen
pip install .
