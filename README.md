pasgen ⚡
Targeted Password Wordlist Generator An optimized Python utility for security researchers to generate targeted wordlists based on known personal information patterns.

🚀 Features
Targeted Patterns: Generates combinations including name+symbol+num, name+num+symbol, num+name+sym, and more.

Multiple Case Styles: Support for UPPERCASE, lowercase, and Capitalized variants.

High Performance: Built using Python's itertools for maximum speed and memory efficiency.

Customizable: Define your own number ranges and symbol sets.

Safe Execution: Processes everything locally with no external data collection.

📥 Installation
Bash
# Clone the repository
git clone https://github.com/AzanChishiya/Pasgen.git

# Enter the directory
cd Pasgen

# Install dependencies
pip install -r requirements.txt
🛠 Usage
Run the script directly using Python:

Bash
python pasgen.py
Follow the interactive prompts to enter names, number ranges, and desired symbols. The resulting wordlist will be saved to your Desktop by default.
